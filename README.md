# CAS
Classroom Assessment System

## 專案目標

建立一套適合國文教學使用的線上評量系統。

第一階段目標：
暑輔期間使用，完成核心測驗流程。

---

## CAS v0.1 功能

### 學生端

- 班級選擇
- 座號＋姓名登入
- 線上測驗
- 題號亂數
- 選項亂數
- 自動批改
- 顯示分數

### 老師端

- 班級管理
- 開始測驗
- 查看學生分數
- 查看完成狀態
- 開放補考

### 統計

- 平均分數
- 字詞王
- 成語王
- 閱讀王

---

## 暑輔測驗流程

前三堂課：

紙本複習與找答案

↓

線上測驗45題

↓

回家複習


第四堂：

複習測驗

↓

未通過者補考
